The engine runs in a Web Worker. Each file is inspected with a timeout; if it hangs or balloons memory (a malicious PDF can drive the parser to multi-GB), the main thread terminates the worker and respawns it — the in-browser equivalent of the per-file process isolation used on the endpoint. Such a file is reported as ESCALATE (isolated).
